# Changelog

All notable changes to SoundBoard will be documented in this file.

---

## v0.2.2 (2026-03-15)

### Bug Fixes
- **播放清單無法加入檔案** — 修復 `sr_cache` 屬性不存在導致加檔失敗
- **快捷鍵無法錄製** — 修復 PySide6 下 `QKeySequence` 建構方式不相容問題
- **音量滑桿無作用** — 修復播放中調整音量不會即時生效的問題，改為即時讀取 item.volume
- **Hold 模式放開不停止** — 新增 `hold_released` 信號，放開滑鼠正確觸發停止
- **缺少 QLineEdit import** — 修復在文字框中按快捷鍵時觸發 NameError
- **儲存時原地修改路徑** — 改為所有檔案複製完成後才批次更新 file_path

### Improvements
- **版面佈局重組** — 左側 CART + 屬性面板，右側播放清單佔滿完整高度，波形圖橫跨底部全寬
- **效能優化** — 波形圖拖曳時不再每幀觸發 properties 面板重建，改為放開滑鼠時才更新
- **Crash Log** — 新增日誌系統，每次啟動產生獨立 log 檔 `logs/soundboard_YYYYMMDD_HHMMSS.log`，含毫秒精度、檔案行號、函數名等詳細資訊，自動保留最近 10 個，閃退時彈出對話框顯示 log 路徑
- **README** — 新增完整功能說明及操作指南

## v0.2.1 (2026-03-15)

### New Features
- **版面改為左右分割** — 廢除 CART/Playlist 切換按鈕，改用 QSplitter 左右並排顯示
- **滑鼠操作邏輯變更** — 左鍵單擊播放（Toggle/Hold），右鍵單擊選取開啟屬性
- **多選改為 Ctrl+右鍵** — 配合右鍵選取，多選操作對應調整
- **音量改為滑桿** — 屬性面板音量從數字輸入框改為 Slider，範圍 0~200%
- **快捷鍵錄製視窗** — 點擊按鈕彈出視窗捕捉按鍵組合
- **獨佔模式 UI 重置** — 被獨佔模式打斷的音軌，UI 狀態立即重置

## v0.2.0 (2026-03-15)

### New Features
- **CART 格網系統** — 自訂行列數量，格子自動縮放填滿畫面
- **播放清單** — 新增、刪除、拖曳排序、命名、自動下一首
- **波形編輯器** — 全曲波形圖、多點音量包絡線、起始/結束標記拖曳
- **音訊引擎** — 單一持續背景混音執行緒，記憶體內即時預轉換 Sample Rate
- **專案管理** — 存檔/另存新檔/開啟舊檔，音檔複製至專案資料夾
- **立體聲音量表** — 綠/黃/紅色段即時顯示
- **全局控制** — 播放/暫停/停止按鈕，ESC 全局停止
- **音效裝置選擇** — 可選輸出裝置及 Buffer Size
- **系統時間顯示** — 綠色大字即時時鐘
